Mozilla Firefox

O Mozilla Firefox é um navegador de internet gratuito, de código aberto e multi-plataforma, desenvolvido pela Mozilla Foundation.

Ele é compatível com os sistemas operacionais Windows, macOS e Linux.

Algumas das características do Firefox são:

  • Não vende dados pessoais para anunciantes
  • Ajuda a proteger as informações pessoais do usuário
  • Bloqueador de popups
  • Pesquisa no Google embutida na barra de ferramentas
  • Integração com RSS
  • Armazenamento automático dos arquivos transferidos no Ambiente de Trabalho
  • Navegação privativa
  • Proteção contra rastreamento

O Firefox é um dos navegadores mais populares da internet.

A Mozilla Foundation é uma organização sem fins lucrativos que tem como missão manter a internet aberta e acessível a todos.
